# Roll To Kill

Currently, a vibed, STS meets Balatro meets dice made in Godot. Some say it's the dark souls of gambling.

## How to Play

1.  Open the project in the Godot Engine.
2.  Run the `main` scene.
3.  Click the "Attack" button to attack the enemy.
4.  Click the "End Turn" button to end your turn.
5.  The enemy will then attack you.
6.  The game ends when either you or the enemy runs out of health.
